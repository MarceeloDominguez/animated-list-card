## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/animated-list-card.git
```

Go to the project directory

```bash
  cd animated-list-card
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run android
```
